IPv4

IPv4（Internet Protocol version 4）是第四版網際網路協定，是目前網際網路上最廣泛使用的 IP 協定版本。IPv4 使用 32 位元位址空間，理論上可提供約 43 億個唯一位址，以四組十進位數字表示，每組範圍為 0-255。

位址格式範例：
- 192.168.1.1（私有網路）
- 8.8.8.8（Google DNS 公有位址）
- 172.16.0.1（私有網路）

IPv4 位址分為五個類別（A、B、C、D、E），其中 A、B、C 類用於一般網路分配，D 類用於多播，E 類保留供實驗使用。由於位址空間有限，現代網路廣泛採用 NAT（網路位址轉譯）技術，允許多個設備共享單一公有 IP 位址。

技術限制與解決方案：
IPv4 位址枯竭問題促使了 IPv6 的發展。目前透過 CIDR（無類別域間路由）、VLSM（可變長度子網遮罩）等技術，可以更有效地利用 IPv4 位址空間。企業和 ISP 也採用私有位址搭配 NAT 的方式，延長 IPv4 的使用壽命。
