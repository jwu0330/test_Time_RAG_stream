IP 位址

IP 位址（Internet Protocol Address）是網際網路協定位址的簡稱，是分配給網路設備的唯一識別碼，用於在網路中定位和識別設備。IP 位址就像是網路世界中的門牌號碼，確保資料能夠正確地從來源傳送到目的地。

IP 位址主要分為兩個版本：IPv4 和 IPv6。IPv4 使用 32 位元位址，以點分十進位表示，例如 192.168.1.1。IPv6 則使用 128 位元位址，以冒號分隔的十六進位表示，例如 2001:0db8:85a3::8a2e:0370:7334。

實際應用場景：
- 家用路由器分配私有 IP 位址給區域網路內的設備
- 網站伺服器使用公有 IP 位址提供網際網路服務
- 企業網路使用 IP 位址進行設備管理和存取控制

技術特點：
IP 位址由網路部分和主機部分組成，透過子網遮罩來區分。每個 IP 位址在其所屬網路中必須是唯一的，以避免位址衝突。現代網路環境中，DHCP 協定常用於自動分配 IP 位址，簡化網路管理工作。
