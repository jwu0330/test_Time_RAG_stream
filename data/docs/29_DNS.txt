DNS

DNS（Domain Name System，網域名稱系統）是網際網路的電話簿，將人類易記的域名（如 www.example.com）轉換為電腦使用的 IP 位址（如 93.184.216.34）。DNS 是網際網路基礎設施的核心組成部分。

基本概念：
DNS 採用分層式架構，從根域（.）開始，到頂級域（.com、.org、.tw），再到二級域（example.com）。每一層由不同的名稱伺服器負責，形成分散式資料庫系統。

查詢流程：
當使用者輸入網址時，瀏覽器首先檢查本地快取。如果沒有記錄，向 DNS 解析器（通常是 ISP 提供）查詢。解析器可能遞迴查詢根伺服器、TLD 伺服器和授權伺服器，最終返回 IP 位址。

實際應用範例：
企業註冊域名 company.com，在 DNS 伺服器上設定 A 記錄指向 Web 伺服器 IP。設定 MX 記錄指向郵件伺服器。使用 CNAME 記錄建立別名，如 www.company.com 指向 company.com。

技術特性：
DNS 使用 UDP 埠 53 進行查詢，TCP 埠 53 用於區域傳輸。支援快取機制，減少查詢次數和延遲。TTL（Time To Live）控制記錄的快取時間。現代 DNS 支援 DNSSEC 提供安全驗證，防止 DNS 欺騙攻擊。
