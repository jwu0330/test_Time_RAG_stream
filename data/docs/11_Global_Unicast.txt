Global Unicast

Global Unicast 位址是 IPv6 中的全球單播位址，相當於 IPv4 的公有位址，可在整個網際網路上路由。這是 IPv6 位址空間中最重要的類型，用於一般的網際網路通訊。

位址範圍與格式：
2000::/3（2000:: 到 3fff:ffff:ffff:ffff:ffff:ffff:ffff:ffff）。目前 IANA 主要分配 2001::/16 範圍內的位址。一個典型的 Global Unicast 位址包含網路前綴、子網 ID 和介面 ID 三個部分。

位址結構範例：
2001:0db8:85a3:0001:0000:8a2e:0370:7334
- 前 48 位元：全球路由前綴（由 ISP 分配）
- 接下來 16 位元：子網 ID（組織內部使用）
- 最後 64 位元：介面 ID（通常由 MAC 位址生成）

實際應用場景：
企業從 ISP 獲得 /48 前綴，可以劃分 65536 個子網。每個子網可容納幾乎無限的設備。網站伺服器、郵件伺服器和其他網路服務使用 Global Unicast 位址提供服務。

技術優勢：
不需要 NAT，每個設備都可以擁有全球唯一的公有位址，簡化端對端通訊。支援更好的安全性和可追溯性。位址空間充足，組織可以靈活規劃網路架構而不用擔心位址耗盡。
