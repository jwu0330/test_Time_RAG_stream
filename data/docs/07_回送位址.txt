回送位址

回送位址（Loopback Address）是一種特殊的 IP 位址，用於設備與自身進行通訊測試。在 IPv4 中，整個 127.0.0.0/8 網段保留作為回送位址，最常用的是 127.0.0.1。在 IPv6 中，回送位址是 ::1。

技術特性：
發送到回送位址的封包不會離開本機，而是在網路堆疊內部迴轉。這使得回送位址成為測試網路軟體和服務的理想工具，無需實際的網路連線。作業系統會建立虛擬的回送介面（如 lo 或 Loopback）來處理這些流量。

實際應用場景：
- 軟體開發：測試網路應用程式時，使用 localhost（127.0.0.1）作為伺服器位址
- 服務測試：驗證 Web 伺服器、資料庫或 API 服務是否正常運作
- 診斷工具：使用 ping 127.0.0.1 測試本機 TCP/IP 堆疊是否正常

管理用途：
網路管理員常用回送位址測試本機網路設定。在路由器和交換機上，回送介面提供穩定的管理位址，不受實體介面狀態影響。許多網路服務預設綁定到回送位址，確保只能從本機存取，提升安全性。
